# Exploratory Data Analysis (EDA) on Titanic Dataset

## Project Overview

This project was completed as **Task 5** of my **Data Analyst Internship**.

The objective of this project is to perform Exploratory Data Analysis (EDA) on the Titanic dataset using Python. Various statistical methods and visualizations were used to identify patterns, relationships, trends, and missing values.

---

## Dataset

- Titanic Dataset
- Rows: 891
- Columns: 12

---

## Objectives

- Explore the dataset
- Analyze missing values
- Generate statistical summaries
- Visualize data using different plots
- Identify relationships between variables
- Summarize important findings

---

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Visualizations

- Histogram
- Boxplot
- Count Plot
- Scatter Plot
- Correlation Heatmap
- Pairplot

---

## Key Findings

- Missing values were present in Age, Cabin, and Embarked columns.
- Most passengers belonged to Passenger Class 3.
- Male passengers were more than female passengers.
- Most passengers were between 20–40 years old.
- More passengers did not survive than survived.
- Cabin contained the highest number of missing values.
- Fare showed a relationship with Passenger Class.

---

## Conclusion

Exploratory Data Analysis helped identify important patterns, trends, and missing values in the Titanic dataset. The project provided valuable insights that can be used for future predictive modeling and machine learning applications.
